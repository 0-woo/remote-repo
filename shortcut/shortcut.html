단축키 설명
alt enter 오류 수정/ 코드 간결화(오류 경고 제안 위 단축키 누르면 수정제안 목록표시)
f2 다음 오류,경고,제안 으로 이동
alt 1 프로젝트창 열고 닫기 프로젝트창 통해 원하는창 검색가능
esc 변경사항을 적용하지 않고 모든 팝업을 닫음/ 에디터로 돌아옴
ctrl e 가장최근 파일 대화상자가 열려 방향키로 파일 탐색
ctrl b 심볼의 선언으로 이동, 필드에서 커서가 필드 선언으로 이동, 클래스 이름에서 클래스 파일로 이동
alt f7 모든 사용위치 표시 / 필드선언또는 인터페이스를 구현한 클래스를 비롯하여 해당 인터페이스가 사용된 모든 위치가 검색창에 표시
ctrl 두번 run anything 열림
ctrl w, ctrl shift w 커서 주변의 코드 섹션을 확장 , 축소
ctrl /, shift ctrl / 그 줄 주석 주석해제, 블록된 모두 주석 주석해제
shift ctrl enter 현재코딩 자동완성 유용함
ctrl alt ㅣ(windows) 간편하게 현재 파일의 서식을 프로젝트 표준에 맞춤
shift ctrl alt t(windows) 모든 리팩토링을 표시
ctrl shift a 모든 액션 검색가능/ 세팅도 가능
shift shift 전체검색
